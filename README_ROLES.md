Kingsday Tester 角色说明（玩家版）

**共通说明**
- Finn 与 游客为必选角色。
- 其余角色可自由加入，人数越多互动越强。
- 抽卡费用与胜利条件为角色专属。

**角色：Finn**
- 初始资源：体力 3，好奇心 2，金钱 0，普通物品 0，橙色物品 0，已佩戴橙色 0。
- 抽卡费用（THEN）：优先金钱 -1，否则好奇心 -1。
- 主动技能：穿上 1 件橙色物品。
- 技能门槛：好奇心 ≥ 2 + 已穿数量，未佩戴橙色 ≥ 1，体力 ≥ 1。
- 技能效果：体力 -1，未佩戴橙色 -1，已佩戴橙色 +1，进度 +1。
- 胜利条件：累计穿上橙色 3 次。
- 规则要点：Finn 被拍照时不可拒绝。

**角色：游客**
- 初始资源：体力 4，好奇心 2，金钱 6，普通物品 1，橙色物品 0，已佩戴橙色 0。
- 抽卡费用（OR）：金钱 -1 或 好奇心 -1。
- 主动技能：发起拍照（目标可拒绝）。
- 拍照门槛：拍照者金钱 ≥1 且体力 ≥1；目标好奇心 ≥2 且拥有橙色物品（Finn 需已佩戴）。
- 拍照效果：拍照者金钱 -1、体力 -1；被拍者金钱 +1；游客进度 +1。
- 胜利条件：拍照次数 ≥5，且独立目标数量满足要求（玩家 ≤3 需 ≥2 名不同目标，玩家 ≥4 需 ≥3 名不同目标）。

**角色：摊主（Vendor）**
- 初始资源：体力 6，好奇心 2，金钱 0，普通物品 2，橙色物品 1，已佩戴橙色 0。
- 抽卡费用（THEN）：优先体力 -1，否则金钱 -1。
- 主动技能：发起交易（目标可拒绝）。
- 交易门槛：卖家好奇心 ≥2、买家好奇心 ≥2、卖家体力 ≥1、买家金钱 > 价格。
- 交易效果：卖家体力 -1、卖家获得金钱、买家付钱得物品，摊主进度 +1。
- 胜利条件：交易次数 ≥（玩家数-1）且不同交易对象 ≥（玩家数-2）。

**角色：食物摊主**
- 初始资源：体力 3，好奇心 1，金钱 0，普通物品 1，橙色物品 0，已佩戴橙色 0。
- 抽卡费用（THEN）：好奇心 -1。
- 主动技能：发起供餐（玩家可选择购买）。
- 供餐门槛：购买者好奇心 ≥2（可能被事件提高），金钱 ≥ 价格（Finn 可被事件设为免费）。
- 供餐效果：购买者体力 +1（可能有额外加成），购买者金钱 -价格，摊主金钱 +价格。
- 供餐成功：达成最低购买人数后记一次成功，摊主进度 +1。
- 胜利条件：成功供餐 ≥3 次，且至少 3 名不同玩家吃到食物。

**角色：表演者**
- 初始资源：体力 6，好奇心 2，金钱 1，普通物品 0，橙色物品 0，已佩戴橙色 1。
- 抽卡费用（OR）：体力 -2 或 体力 -1 且好奇心 -1。
- 主动技能：发起表演（围观者选择收益）。
- 表演门槛：体力 ≥2 才能开始（否则可请求志愿者）。
- 表演成功：围观成功人数满足要求后记一次成功，表演者进度 +1。
- 体力消耗：成功 -2，失败 -1（部分事件可免体力）。
- 胜利条件：成功表演 ≥3 次。

**角色：志愿者**
- 初始资源：体力 2，好奇心 2，金钱 5，普通物品 1，橙色物品 0，已佩戴橙色 0。
- 抽卡费用（THEN）：优先金钱 -1，否则体力 -1。
- 主动技能：无（通过帮助他人获得进度）。
- 可帮助行为类型：拍照、交易、Finn 穿橙、供餐、表演。
- 胜利条件：帮助 3 种不同类型的行为。
